 Project Description
This project detects whether an SMS message is Spam or Not Spam using Machine Learning and NLP.

Tech Stack:
- Python
- Streamlit
- Scikit-learn
- NLTK
- Pandas
- NumPy

 How it works
1. Text preprocessing (cleaning, stopwords removal)
2. TF-IDF vectorization
3. Machine learning model prediction
4. Streamlit web interface

Files in this project
- app.py → Streamlit app
- model.pkl → trained model
- vectorizer.pkl → TF-IDF model
- requirements.txt → dependencies

Features
- Predict spam or ham messages
- Simple UI
- Fast predictions
- Live deployed app
